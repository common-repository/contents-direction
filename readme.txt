== Description ==

This plugin used to change the direction of the post content RTL by adding a metabox to every post admin page.

You will find a dropdown select box with title ( direction of the content ), you can choose weather the content RTL or LTR .

( please refer to the screenshot section to clarify more ) .

A few notes about the section above:

* RTL == Right to left
* LTR  == Left to right


== Installation ==

1. Upload `content-direction` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Turn to any post or page, you will find the post direction metabox
